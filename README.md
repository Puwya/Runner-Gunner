Game in development

*I can only work on it when I have free time from school*

## How to install

**MAKE SURE YOU HAVE SDL INSTALLED!**

>1. Clone Repo
>
>2. mkdir build
>
>3. cd build
>
>4. cmake ..
>
>5. ./main
